#ZeroLeak
ZeroLeak is a Chrome extension to check sensitive files.

check for backup file:

* web.(zip|rar|tar.gz...)
* wwwroot.(zip|rar|tar.gz...)
* %hostname%.(zip|rar|tar.gz...)

recursively check(depth is 3):

* .svn/entries
* .git/config
* robots.txt
* .DS_Store
* %folder%.(zip|rar|tar.gz...)

You need to pack it yourself.
